# Contributing to AI Agent Sketchbook

We are excited that you want to contribute to the **AI Agent Sketchbook**! Whether you're fixing a bug, adding a new template, or improving documentation, your help is appreciated.

## 🛣️ How to Contribute

1. **Fork the Repository**: Create your own copy of the project.
2. **Create a Branch**: Use a descriptive name for your branch (e.g., `feat/new-rag-template`).
3. **Make Your Changes**: Ensure your code follows the project's style and includes necessary documentation.
4. **Submit a Pull Request**: Describe your changes and why they are beneficial.

## 📝 Contribution Areas

- **Templates**: New starter kits for emerging agent architectures.
- **Patterns**: Detailed explanations of agentic behaviors.
- **Sketches**: Architecture diagrams or UI/UX mockups for agents.
- **Docs**: Improving existing guides or adding new ones.

## ⚖️ Code of Conduct

Please be respectful and professional in all interactions within this project.
