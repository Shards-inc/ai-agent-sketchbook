# Short-Term Memory and Context Management

**Short-term memory** in AI agents refers to the immediate, transient information that an agent uses during a single interaction or a short sequence of interactions. This is primarily managed within the **context window** of the Large Language Model (LLM) and is crucial for maintaining coherence, understanding recent turns in a conversation, and executing multi-step tasks.

## 🧠 Concept

Unlike long-term memory, which is persistent and externalized, short-term memory is ephemeral. It encompasses the current prompt, recent conversational history, and any intermediate thoughts or observations generated by the agent. The primary challenge with short-term memory is the **finite context window size** of LLMs, which limits how much information can be held at once.

Effective short-term memory management involves strategies to:

-   **Maintain Coherence**: Ensure the agent's responses are relevant to the ongoing conversation.
-   **Track State**: Keep track of current goals, sub-tasks, and relevant variables.
-   **Prioritize Information**: Decide which parts of the history are most important to keep within the limited context.

## 📝 Management Strategies

Several strategies are employed to manage short-term memory within the constraints of the context window:

1.  **Fixed Window**: Keep only the `N` most recent turns of a conversation. This is simple but can lead to loss of important information from earlier in the dialogue.

2.  **Summarization**: Periodically summarize older parts of the conversation or agent's internal monologue. The summary replaces the detailed history, preserving key information while reducing token count.

3.  **Embedding-based Retrieval (Hybrid Approach)**: Use embeddings to identify and retrieve the most semantically relevant past interactions or internal thoughts from a larger history, effectively bridging short-term and long-term memory concepts.

4.  **Prompt Engineering**: Structure prompts to explicitly guide the LLM on what information to prioritize and how to use the provided context.

5.  **Hierarchical Memory**: Combine summarization with a fixed window, where recent turns are kept verbatim, and older turns are summarized.

## 🚀 Implementation Example (Conceptual)

```python
from typing import List, Dict, Any

class MockLLM:
    def generate(self, prompt: str) -> str:
        # Simulate LLM response based on prompt content
        if "summarize" in prompt.lower():
            return "Summary: User discussed initial problem, then asked for clarification on solution steps."
        return f"LLM processed: {prompt[:50]}..."

class ShortTermMemoryManager:
    """Manages the short-term memory (context window) for an agent."""
    def __init__(self, llm: MockLLM, max_tokens: int = 1000):
        self.llm = llm
        self.conversation_history: List[Dict[str, str]] = []
        self.max_tokens = max_tokens # Simulated max tokens for context window

    def add_message(self, role: str, content: str):
        self.conversation_history.append({"role": role, "content": content})
        self._prune_history()

    def _calculate_tokens(self, messages: List[Dict[str, str]]) -> int:
        # A very simplified token calculation. In reality, this would use a tokenizer.
        return sum(len(msg["content"].split()) for msg in messages) # Count words as tokens

    def _prune_history(self):
        """Prunes the conversation history to fit within max_tokens using summarization."""
        current_tokens = self._calculate_tokens(self.conversation_history)

        while current_tokens > self.max_tokens and len(self.conversation_history) > 1:
            # Try to summarize older messages if they are not the very first one
            oldest_message = self.conversation_history[0]
            if oldest_message["role"] == "system" or oldest_message["role"] == "user":
                # Summarize the oldest user/assistant turn
                messages_to_summarize = self.conversation_history[0:2] # Take first user and agent response
                combined_text = " ".join([msg["content"] for msg in messages_to_summarize])
                summary_prompt = f"Summarize the following conversation for context: {combined_text}"
                summary = self.llm.generate(summary_prompt)

                # Replace the summarized messages with the summary
                self.conversation_history = [{
                    "role": "system",
                    "content": f"Previous conversation summary: {summary}"
                }] + self.conversation_history[2:]
            else:
                # If it's already a summary or system message, just remove the oldest
                self.conversation_history.pop(0)

            current_tokens = self._calculate_tokens(self.conversation_history)
            print(f"History pruned. Current tokens: {current_tokens}")

    def get_context(self) -> List[Dict[str, str]]:
        """Returns the current conversation history as context for the LLM."""
        return self.conversation_history

if __name__ == "__main__":
    mock_llm = MockLLM()
    memory_manager = ShortTermMemoryManager(mock_llm, max_tokens=50) # Small context for demonstration

    print("Adding initial messages...")
    memory_manager.add_message("user", "Hi, I need help with my project. It involves data analysis.")
    memory_manager.add_message("assistant", "Okay, I can help with data analysis. What kind of data are you working with?")
    memory_manager.add_message("user", "I have a CSV file with sales data from the last quarter.")
    memory_manager.add_message("assistant", "Great. Can you tell me about the columns in the CSV and what you want to achieve?")

    print("\nCurrent Context:")
    for msg in memory_manager.get_context():
        print(f"  {msg["role"]}: {msg["content"]}")

    print("\nAdding more messages to trigger pruning...")
    memory_manager.add_message("user", "The columns are ProductID, SalesAmount, Date, and Region. I want to see total sales per region.")
    memory_manager.add_message("assistant", "Understood. I will write a Python script to calculate total sales per region from your CSV.")

    print("\nFinal Context after pruning:")
    for msg in memory_manager.get_context():
        print(f"  {msg["role"]}: {msg["content"]}")
```
